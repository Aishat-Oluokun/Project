# HR Analytics Dashboard using Microsoft Excel
## Project Overview
Every organisation depends on accurate data to make a good decisions. In this project,iI worked with a raw HR dataset containing employee records, department informaton,hire date, salary, employment status, and performance scores.
Before the analysis the data set was cleaned and transfromed to ensure consistency and accuracy. After cleaning and preparing the dataset then i create a pivot table and chart so as to create an interactive dashboard to answer key business questions and provide insight from the dataset.
My objective was to clean and transform the raw data into reliable dataset and build an interactive dashboard that answer important business question related to employees, salary performance of employees and attrition.
##About the dataset
The dataset include the employee records,which include
- Employee ID
- Employee Name
- Department Code
- Hire Date
- Performance Scores
- Salary
- Employment Status
- Department Name
- performance Band
- Eligible Bonus Amount
- Year of Service
  The original dataset consist of inconsistencies such as duplicate records, inconsistence formating and mixed data types.
##Data cleaning Process
  Before the dataset was used for analysis, it was cleaned as follows:
-I removed duplicate records.7 duplicate values was found and removed;150 unique values remain.This count include both empty cells and space.
-I removed unecessary empty cell
-I converted the salary values into numeric format
-I created a Full Name column
-I trimed unecessary space in each column
-I convert the date column to date format
-I created more column using formulas like xlookup,vlookup and dateif to calculate year of services. which include
-Department Name
-Performance Band
- Eligible Bonus Amount
- Then i calculated the year of service
  At the end of the cleaning processed the cleaned data was moved together on a side of the excel sheet so as to change it to table format for pivot table. This is done because the original data can not be replaced by the cleaned data so as to show the processed used for cleaning.
##Turning Data into insight
-The cleaned dataset was used to built pivotTables and PivotChart to answer HR business questions.
The dashboard highlights the KPIs and the visualizations:
The KPIs include
-Total Employees
-Total payroll
-Average salary
-Attririon Rate
The visualizations include;
-Salary Distribution by performance band:This shows the salary paid to employee based on their performannce band. 
-Employment status by department:THis compare the employment status either Active,Left or Resigned employee across department
-Total Bonus by Department: This shows the total Bonus recived by employee in each department
-Average Salary by Department:This compares the average slalries recived by employee across Department
-Employment Statsus Distribution:This shows the number of each employment status in which active has the highest number
-Hiring Trend by Month:This shows the monthly trend of employment of employee
-Attrition Rate by Department: This show the turnover of employee across Department
-Bonus Payment by Performance Band:This shows the bonus given to employee based on their performance band.
##Key Insight
The dashboard provide several information and clear insight of compensation and performance of employee
-The salary is compared across department and performance band
-The bonus is compared across performance band and department
-Hiring is compared across year and months.
-Attrition analysis provide information about department that need improvement
